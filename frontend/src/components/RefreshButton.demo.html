⚙️ JavaScript Handler

The button uses a simple click handler to add the spinning animation:

button.addEventListener('click', async () => { // Add spinning class to trigger animation button.classList.add('spinning'); button.disabled = true; try { // Perform your async operation await fetchData(); } finally { // Remove spinning animation button.classList.remove('spinning'); button.disabled = false; } });
