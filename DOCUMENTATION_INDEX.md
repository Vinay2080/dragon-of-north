## 🎨 Design System

### Colors

- **Accent**: Indigo (#4f46e5)
- **Borders**: Slate-200 (#e2e8f0)
- **Backgrounds**: White (#ffffff) + Light grays
- **Text**: Dark slate-900 / slate-600

### Depth

- **Shadows**: Two-layer system (1px near + 12px far)
- **Elevation**: Subtle, not heavy
- **Glow**: Removed from light mode

### Typography

- **Hierarchy**: Clear primary → secondary → tertiary
- **Contrast**: WCAG AA compliant
- **Sizing**: Standard Tailwind scale
